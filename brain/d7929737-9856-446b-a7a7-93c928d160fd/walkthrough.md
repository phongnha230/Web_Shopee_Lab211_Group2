# Authentication Flow Implementation

I have implemented the Login and Register pages matching the provided designs and updated the main navigation to link to them.

## Changes

### 1. Header Navigation
- Updated the **User Icon** in the header to link directly to `/login`.

### 2. Login Page (`/login`)
- Implemented a minimalist, centered login form.
- Features:
    - "Welcome Back" heading.
    - Email & Password inputs.
    - Social Login buttons (Google, Apple).
    - Link to "Create account".

### 3. Register Page (`/register`)
- Implemented a split-layout registration page.
- Features:
    - **Left Side**: "Join the Nest" registration form (Name, Email, Password).
    - **Right Side**: Full-height feature image (Streetwear lifestyle).
    - Link back to "Log In".
